# The Physics Law

## N(t) = N₀ × e^(-λt)

Where:
- N(t) = number of particles remaining at time t
- N₀ = initial number of particles (e.g., 1000)
- λ = decay constant (how fast particles decay)
- e = Euler's number (2.718...)

----


## Half-life: Time it takes for half the particles to decay

### t½ = ln(2) / λ ≈ 0.693 / λ

## Example:

- Start with: 1000 particles
- λ = 0.1 per second
- Half-life = 6.93 seconds
- After 6.93 seconds: ~500 particles remain
- After 13.86 seconds: ~250 particles remain
